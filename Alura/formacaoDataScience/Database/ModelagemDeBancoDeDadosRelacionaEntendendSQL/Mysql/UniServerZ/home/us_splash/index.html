The Uniform Server | Downloads | Wiki | Support Forum

The Uniform Server is a WAMP package that allows you to run a server on any XP, Vista, W7, W8 or W10 Windows OS based computer (UniServer ZeroXIV requires W8+). It is small and quick to download, and can be easily moved around. It also can be setup and used as a production/live server. Developers can use The Uniform Server to test their applications which require PHP, MySQL and the Apache HTTPd Server.
