General

The following are common to installing applications:

MySQL and MariaDB

MariaDB is a drop in replacement for MySQL and instructions for installing applications on either server are the same. Where applicable substitute MariaDB for MySQL.

Host

During installation when requested for a host name do not use localhost instead use 127.0.0.1

e-mail address

Applications requiring you to enter your e-mail address use the e-mail address you set as default when configuring MSMTP, for details see Edit MSMTP Configuration

WordPress

WordPress is a free and open source application designed primarily for blogging and creating websites. It is a content-management system (CMS) based on PHP and MySQL the following page provides detailed information for installing on Uniform Server Zero.

MediaWiki

MediaWiki is a free and open source wiki software, used to power wiki websites. It is based on PHP and MySQL the following page provides detailed information for installing on Uniform Server Zero.
